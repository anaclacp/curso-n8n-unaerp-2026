## Como exportar esses fluxos

### Para exportar esses fluxos para o seu n8n, você pode apenas copiar o json de cada fluxo e colar no seu canvas do n8n. 

Para isso:
1. Abra o arquivo do fluxo que você quer usar (ex: `fluxos/fluxo-1.json`).
2. Copie todo o conteúdo do arquivo (Ctrl + A, Ctrl + C).
3. Vá para o seu n8n, crie um novo fluxo e cole o conteúdo (Ctrl + V).
4. O n8n vai reconhecer o formato e criar os nós e conexões automaticamente.
